# mountedで画面が表示されてことをイベントで感知する

一覧画面（Home）に表示させる

->Homeのmountedに記述すると思う！！

## やること

- ==と===の違う点
  -> == 等価演算子：値+型が等価であるか。文字列と数値の比較の場合、文字列を数値に変換してくれる
  -> == === 厳密等価演算子：値が等価であるか。文字列は数値に変換されない
  
- ヘッダタブを実装し、画面を切り替えて値を受け渡す
- 余裕があれば、画面間の受け渡し
